RVAF-GUI
===
MFC GUI Interface to drive [__RVAF__](https://github.com/P-Chao/RVAF)
![](img/rvaf-gui.png)

* Left Zoon: Set parameter of algorithm
* Middle Zoon: Function Button
* Right Zoon: Visual Output
